# Настройка ПК для локального мессенджера

Пошаговая инструкция: что установить и как работать со мной (Cursor).

---

## Что нужно установить

| Что | Зачем |
|-----|--------|
| **Node.js (LTS)** | Запуск сервера мессенджера, фронт или десктоп (Electron), установка пакетов через npm. |
| **Git** | Версии кода, откат изменений, удобно при сдаче заданий. Можно обойтись без него. |

**База данных** отдельно ставить не нужно: будем использовать **SQLite** — это один файл на диске, библиотека подключается через npm.

---

## Шаг 1: Установка Node.js

1. Скачай установщик: https://nodejs.org/ — выбери **LTS** (рекомендуемая версия).
2. Запусти установщик, пройди шаги. Важно: отметь пункт **"Add to PATH"** (добавить в PATH).
3. После установки **закрой и заново открой** Cursor (или терминал), чтобы подхватился PATH.
4. Проверка: в терминале Cursor выполни `node -v` и `npm -v` — должны показаться версии.

**Через winget (если удобнее):**
```powershell
winget install OpenJS.NodeJS.LTS --accept-package-agreements
```
После этого закрой и открой Cursor.

---

## Шаг 2: Установка Git (по желанию)

1. Скачай: https://git-scm.com/download/win  
2. Установи с настройками по умолчанию.  
3. Проверка: `git --version`.

**Через winget:**
```powershell
winget install Git.Git --accept-package-agreements
```

---

## Шаг 3: Как это связано со мной (Cursor)

- Ты открываешь папку **`local mobile messenger`** в Cursor как проект (File → Open Folder).
- Я вижу все файлы в этой папке и могу создавать и править код, предлагать команды для терминала.
- Ничего дополнительно «подвязывать» не нужно: просто пиши задание в чат, например: «сделай сервер на Node и простой чат в браузере» или «добавь сохранение сообщений в SQLite».
- Терминал в Cursor — тот же PowerShell/CMD на твоём ПК; команды вроде `npm install` или `node server.js` я могу предлагать, ты их запускаешь (или разрешаешь мне выполнять).

**Перед первым заданием:**  
Убедись, что после установки Node в терминале Cursor работают команды `node -v` и `npm -v`. Если да — можно давать задание на мессенджер.
